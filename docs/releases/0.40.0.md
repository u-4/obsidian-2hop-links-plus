# 2Hop Links Plus 0.40.0

[日本語](#日本語) | [English](#english)

## 日本語

このリリースは、大きな Vault の起動時とタブ切替時に重複していた計算を減らします。

- Obsidian のレイアウト復元後に1.5秒待ってから初回計算を開始
- 短時間のタブ切替は最後の要求だけを実行（既定200ms、設定で変更可能）
- リンクグラフ、直近ノートの表示結果、Canvasリンク索引を再利用
- 古くなった計算を中止し、同時に始まった同一処理を共有
- `Related, Cosense-like` では表示に不要な PageRank 計算を省略
- 性能統計を確認・リセットするコマンドを追加
- 依存関係を増やさず、合成Vaultによる自動性能・競合テストを追加

関連度順を含む既存のソート、検索、行ジャンプ、Canvas対応は維持されます。
更新前に既存のプラグインフォルダをバックアップしてください。問題がある場合は、
バックアップした `main.js`、`manifest.json`、`styles.css` を戻して再読み込みできます。

## English

This release reduces duplicated startup and tab-switch work in large Vaults.

- Starts the first calculation after workspace restoration and a 1.5-second grace period
- Coalesces rapid tab changes into the latest request (200 ms by default, configurable)
- Reuses the link graph, recent note results, and the Canvas link index
- Cancels superseded work and shares identical in-flight calculations
- Skips unused PageRank work for `Related, Cosense-like`
- Adds commands to inspect and reset performance statistics
- Adds deterministic synthetic-Vault performance and coordination tests without new dependencies

Existing relevance sorting, search, link-line navigation, and Canvas behavior are
preserved. Back up the existing plugin directory before updating. To roll back,
restore `main.js`, `manifest.json`, and `styles.css`, then reload the plugin.
