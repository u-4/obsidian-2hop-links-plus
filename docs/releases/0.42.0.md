# 2Hop Links Plus 0.42.0

[日本語](#日本語) | [English](#english)

## 日本語

このリリースは、iPhoneや狭い分割ペインでの操作性とカードの読みやすさを改善し、
本文カードと関連リンク領域をテーマCSSから安全に分けられる連携境界を追加します。

- 検索・設定・一時ソートを、画面幅にかかわらず一行に配置
- 検索アイコンを押した時だけ、同じ長方形の操作面の中で検索欄を展開
- 狭い画面では44 pxの操作領域と読みやすい2列カードを使用
- 検索で一致件数が変わっても、検索開始前の結果領域を保ってツールバーの上下移動を防止
- 一時ソートをチェック付きメニューへ変更し、設定の既定値と異なる場合だけアクセント色の点を表示
- ライト／ダークテーマに合わせてソートメニューの背景色と文字色を明示
- 本文内2-hop領域に安定したクラスと配置情報を追加し、任意のCSSから本文カード、
  2-hop、Obsidian標準backlinksの境界を調整可能
- activeなseparate paneを閉じて本文内表示へ戻す際、同じworkspace領域のMarkdownだけを
  安全に復元し、Obsidianが別のタブを選んだ場合はその選択を尊重

外部のCosense風CSSはこのプラグインに同梱されず、プラグイン単体でも動作します。
依存関係と最小対応Obsidianバージョン（1.3.5）は変更していません。

更新前に既存のプラグインフォルダをバックアップしてください。問題がある場合は、
バックアップした `main.js`、`manifest.json`、`styles.css` を戻してObsidianを再読み込みできます。

## English

This release improves toolbar usability and card readability on iPhone-sized and
narrow split views, while adding a stable theme-CSS boundary between the note card
and document-related regions.

- Keeps search, settings, and temporary sorting on one row at every pane width
- Expands search inside the same rectangular control only after its icon is activated
- Uses 44 px controls and a readable two-column card grid in narrow layouts
- Preserves the pre-filter result height so changing the match count does not move the toolbar
- Replaces the temporary sort select with a checked menu and shows an accent dot only for an override
- Gives the sort menu explicit readable light- and dark-theme colors
- Adds stable classes and placement data for optional CSS to separate the note card,
  inline 2-hop results, and Obsidian's standard backlinks
- Restores a Markdown leaf from the same workspace container when an active separate
  pane closes, while respecting another leaf already selected by Obsidian

The optional Cosense-style CSS is not bundled or required. Dependencies and the
minimum supported Obsidian version (1.3.5) are unchanged.

Back up the existing plugin directory before updating. To roll back, restore
`main.js`, `manifest.json`, and `styles.css`, then reload Obsidian.
