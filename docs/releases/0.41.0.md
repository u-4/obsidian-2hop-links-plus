# 2Hop Links Plus 0.41.0

[日本語](#日本語) | [English](#english)

## 日本語

このリリースは、長文の Markdown ノート内でノート本文と 2-hop 表示を行き来しやすくし、ツールバーと一時ソートの操作を安定させます。

- Markdown ノートの右上に、ノート最上部と本文内の 2-hop 表示を往復する円形の二重矢印ボタンを追加
- Live Preview と Reading view の両方に対応し、現在位置に応じて移動先の説明を切り替え
- 「視差効果を減らす」が有効な場合は即時移動を使用
- 自動位置調整中にホイール、タッチ、ポインター、キー入力があった場合は利用者の操作を優先
- タブ、分割、表示モードを切り替えてもボタンが重複・残留しないよう後片付けを強化
- 検索欄を利用可能な幅まで広げ、設定ボタンと一時ソートを右側にまとめたレスポンシブなツールバーへ変更
- separate pane の一時ソートを、検索内容とペインを保ったままその場で更新
- 自動読み込みを無効にして手動表示したカードも、一時ソート後に表示を維持

右上の移動ボタンは本文内表示専用で、separate pane には表示されません。依存関係と最小対応 Obsidian バージョン（1.3.5）は変更していません。

更新前に既存のプラグインフォルダをバックアップしてください。問題がある場合は、バックアップした `main.js`、`manifest.json`、`styles.css` を戻して Obsidian を再読み込みできます。

## English

This release makes it easier to move between a long Markdown note and its inline 2-hop results, while improving toolbar and temporary-sort stability.

- Adds a circular double-chevron button in the upper-right of Markdown note headers for moving between the note top and inline 2-hop results
- Supports both Live Preview and Reading view, with destination text that follows the current position
- Uses immediate movement when reduced motion is enabled
- Gives wheel, touch, pointer, and keyboard input priority over automatic alignment retries
- Strengthens cleanup so the control is not duplicated or left behind across tabs, splits, modes, and non-Markdown views
- Expands search into available space and groups settings and temporary sorting on the right in a responsive toolbar
- Updates temporary sorting in a separate pane in place, preserving its search text and pane state
- Keeps manually shown cards visible after temporary sorting when auto-load is disabled

The upper-right navigation button is only for inline results and is not shown in a separate pane. Dependencies and the minimum supported Obsidian version (1.3.5) are unchanged.

Back up the existing plugin directory before updating. To roll back, restore `main.js`, `manifest.json`, and `styles.css`, then reload Obsidian.
